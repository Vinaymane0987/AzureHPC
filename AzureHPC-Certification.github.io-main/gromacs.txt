        <p style="margin-top: 0pt; margin-bottom: 0pt; text-indent: 28.8pt; font-size: 11pt;"><span style="font-family: Calibri;">GROMACS is a versatile package to perform molecular dynamics, i.e. simulate the Newtonian equations of motion for systems with hundreds to millions of particles.</span> <span style="font-family: Calibri;">It is primarily designed for biochemical molecules like proteins, lipids and nucleic acids that have a lot of complicated bonded interactions, but since GROMACS is extremely fast at calculating the nonbonded interactions</span><span style="font-family: Calibri;">.</span></p>
        <p style="margin-top: 0pt; margin-bottom: 0pt; font-size: 11pt;"><span style="font-family: Calibri;">GROMACS provides</span><span style="font-family: Calibri; font-style: italic;">extremely high performance</span><span style="font-family: Calibri;">compared to all other programs</span><span style="font-family: Calibri;">.</span></p>
        <p style="margin-top: 0pt; margin-bottom: 0pt; font-size: 11pt;"><span style="font-family: Calibri;">GROMACS is user-friendly, with topologies and parameter files written in clear text format.</span> <span style="font-family: Calibri;">There is a lot of consistency checking, and clear error messages are issued when something is wrong</span><span style="font-family: Calibri;">.</span> <span style="font-family: Calibri;">Both run input files and trajectories are independent of hardware endian-ness, and can thus be read by any version GROMACS, even if it was compiled using a different floating-point precision.</span> <span style="font-family: Calibri;">GROMACS contains several state-of-the-art algorithms that make it possible to extend the time steps is simulations significantly, and thereby further enhance performance without sacrificing accuracy or detail.</span></p>
        <p style="margin-top: 0pt; margin-bottom: 0pt; font-size: 11pt;"><span style="font-family: Calibri;">M</span><span style="font-family: Calibri;">any groups are also using it for research on non-biological systems, e.g. polymers.</span></p>
