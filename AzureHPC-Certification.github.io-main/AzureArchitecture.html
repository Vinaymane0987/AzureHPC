Deploying a Virtual Machine on Azure Cloud Platform

AZURE VIRTUAL MACHINE (VM)

A VM is a virtualized instance of a computer that can perform almost all the same functions as a computer, including running applications and operating systems.

 

Things to be considered while deploying a Virtual Machine on Azure cloud platform,

  1. The Prerequisites required for the application to be installed
  2. The Azure region where the resources will be deployed
  3. The size and configuration of the VM
  4. The operating system required for the application
  5. The compatible drivers for the given application

 

Depending on the workload, the user can choose the appropriate VM size for the given VM series. For a complete list of available VM’s, please check here.
