    <li style="margin-left: 27.6pt; text-align: justify; padding-left: 8.4pt; font-family: serif; font-size: 10pt; color: #171717; background-color: #ffffff; -aw-font-family: 'Symbol'; -aw-font-weight: normal; -aw-number-format: '';"><span style="font-family: 'Segoe UI'; font-size: 12pt;">All external vehicle aerodynamics models</span> <span style="font-family: 'Segoe UI'; font-size: 12pt;">tested </span><span style="font-family: 'Segoe UI'; font-size: 12pt;">demonstrated good </span><span style="font-family: 'Segoe UI'; font-size: 12pt;">CPU acceleration when running in multi-node configuration</span><span style="font-family: 'Segoe UI'; font-size: 12pt;">s</span><span style="font-family: 'Segoe UI'; font-size: 12pt;">.</span></li>
    <li style="margin-left: 27.6pt; text-align: justify; padding-left: 8.4pt; font-family: serif; font-size: 10pt; color: #171717; background-color: #ffffff; -aw-font-family: 'Symbol'; -aw-font-weight: normal; -aw-number-format: '';"><span style="font-family: 'Segoe UI'; font-size: 12pt;">The meshing, setup and solver applications in </span><span style="font-family: 'Segoe UI'; font-size: 12pt;">ELEMENTS</span><span style="font-family: 'Segoe UI'; font-size: 12pt;"> can all be run in parallel, </span><span style="font-family: 'Segoe UI'; font-size: 12pt;">thus </span><span style="font-family: 'Segoe UI'; font-size: 12pt;">making this CFD tool ideal for execution in multi-node configurations (no need for mesh decomposition/reconstruction).</span></li>
