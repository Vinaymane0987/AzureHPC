    <p style="margin-top: 0pt; margin-bottom: 14pt; text-align: justify; line-height: 115%; font-size: 11pt; background-color: #ffffff;"><span style="font-family: 'Segoe UI'; color: #333333; background-color: #ffffff;">Devito is a </span><span style="font-family: 'Segoe UI'; color: #333333; background-color: #ffffff;">functional language,</span> <span style="font-family: 'Segoe UI'; color: #333333; background-color: #ffffff;">implemented as a </span><span style="font-family: 'Segoe UI'; color: #333333; background-color: #ffffff;">Python package</span><span style="font-family: 'Segoe UI'; color: #333333; background-color: #ffffff;">,</span> <span style="font-family: 'Segoe UI'; color: #333333; background-color: #ffffff;">which </span><span style="font-family: 'Segoe UI'; color: #333333; background-color: #ffffff;">enables </span><span style="font-family: 'Segoe UI'; color: #333333; background-color: #ffffff;">optimized stencil computation (e.g., finite differences, image processing, machine learning) from high-level symbolic problem definitions. Devito builds on </span><a style="text-decoration: none;" href="https://www.sympy.org/"><span style="font-family: 'Segoe UI'; text-decoration: underline; color: #337ab7; background-color: #ffffff;">SymPy</span></a><span style="font-family: 'Segoe UI'; color: #333333; background-color: #ffffff;"> and employs automated code generation and just-in-time compilation to execute optimized computational kernels on several computer platforms, including CPUs, GPUs, and clusters thereof.</span><span style="font-family: 'Segoe UI'; color: #333333; background-color: #ffffff;"> For more information, please visit the link </span><a style="text-decoration: none;" href="https://www.devitoproject.org/"><span style="font-family: 'Segoe UI'; text-decoration: underline; color: #0000ff; background-color: #ffffff;">https://www.devitoproject.org/</span></a></p>
    <p style="margin-top: 0pt; margin-bottom: 0pt; line-height: 115%; font-size: 11pt; background-color: #ffffff;"><span style="font-family: 'Segoe UI'; color: #333333;">Symbolic computation is a powerful tool that allows users to:</span></p>
