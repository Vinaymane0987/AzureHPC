<p><h2 style="margin-top: 2pt; margin-left: 28.8pt; margin-bottom: 0pt; text-indent: -28.8pt; page-break-inside: avoid; page-break-after: avoid; line-height: 108%; font-size: 12pt;"><span style="font-family: 'Calibri Light'; font-size: 13pt; font-weight: normal; color: #2f5496;">2.2</span> <a name="_Toc107417806"></a><span style="font-family: 'Calibri Light'; font-weight: normal; color: #2f5496;">Azure Virtual Machine (VM)</span></h2> </p>
<p style="margin-top: 0pt; margin-bottom: 8pt; text-indent: 28.8pt; text-align: justify; line-height: 108%; font-size: 11pt;"><span style="font-family: Calibri;">A VM is a virtualized instance of a computer that can perform almost </span><span style="font-family: Calibri;">all of</span><span style="font-family: Calibri;"> the same functions as a computer, including running applications and operating systems.</span></p>
<p style="margin-top: 0pt; margin-bottom: 8pt; text-align: justify; line-height: 108%; font-size: 11pt;"><span style="font-family: Calibri;">An Azure VM gives you the flexibility of virtualization without having to buy and maintain the physical hardware that runs it. However, you still need to maintain the VM by performing tasks, such as configuring, patching, and installing the software that runs on it.</span></p>
<p style="margin-top: 0pt; margin-bottom: 8pt; text-align: justify; line-height: 108%; font-size: 11pt;"><span style="font-family: Calibri;">Things to be considered before deploying a VM,</span></p>

<li style="margin-left: 64.06pt; text-align: justify; padding-left: 7.94pt; font-family: serif; font-size: 11pt; color: #171717; background-color: #ffffff; -aw-font-family: 'Symbol'; -aw-font-weight: normal; -aw-number-format: '';"><span style="font-family: Calibri;">The location where the resources are stored</span></li>
<li style="margin-left: 64.06pt; text-align: justify; padding-left: 7.94pt; font-family: serif; font-size: 11pt; color: #171717; background-color: #ffffff; -aw-font-family: 'Symbol'; -aw-font-weight: normal; -aw-number-format: '';"><span style="font-family: Calibri;">The size of the VM</span></li>
<li style="margin-left: 64.06pt; padding-left: 7.94pt; font-family: serif; font-size: 11pt; color: #171717; background-color: #ffffff; -aw-font-family: 'Symbol'; -aw-font-weight: normal; -aw-number-format: '';"><span style="font-family: Calibri;">The operating system that the VM runs</span></li>
<li style="margin-left: 64.06pt; padding-left: 7.94pt; font-family: serif; font-size: 11pt; color: #171717; background-color: #ffffff; -aw-font-family: 'Symbol'; -aw-font-weight: normal; -aw-number-format: '';"><span style="font-family: Calibri;">The configuration of the VM after it starts</span></li>
<li style="margin-left: 64.06pt; padding-left: 7.94pt; font-family: serif; font-size: 11pt; color: #171717; background-color: #ffffff; -aw-font-family: 'Symbol'; -aw-font-weight: normal; -aw-number-format: '';"><span style="font-family: Calibri;">The related resources that the VM needs</span></li>
