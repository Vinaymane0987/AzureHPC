        <h2 style="margin-top: 2pt; margin-left: 43pt; margin-bottom: 0pt; text-indent: -28.8pt; page-break-inside: avoid; page-break-after: avoid; font-size: 13pt;"><span style="font-family: Calibri; font-weight: normal; color: #2f5496;">5.1</span> <a name="_Toc81323178"></a><a name="_Toc81330560"></a><a name="_Toc81390112"></a><a name="_Toc81411725"></a><a name="_Toc81413122"></a><a name="_Toc98784281"></a><span style="font-family: Calibri; font-weight: normal; color: #2f5496;">Samadii-</span><span style="font-family: Calibri; font-weight: normal; color: #2f5496;">Dem</span><span style="font-family: Calibri; font-weight: normal; color: #2f5496;"> overview:</span></h2>
        <p style="margin-top: 0pt; margin-bottom: 0pt; text-indent: 36pt; line-height: 125%; font-size: 11pt;"><span style="font-family: Calibri; color: #1d2129; background-color: #ffffff;">There exist several types of solid particles in the industrial fields such as: in terms of particle-scale, from Nano-sized particles which cause a defect in the semiconductor process, virus, micron-particles which are used in the printer-toner, chemical by-products, ultra-fine powder in the grinding process, sand, gravel, grains, and mineral ores.</span></p>
        <p style="margin-top: 0pt; margin-bottom: 0pt; text-indent: 36pt; line-height: 125%; font-size: 11pt;"><span style="font-family: Calibri; color: #1d2129; background-color: #ffffff;">Discrete element method (DEM), also called as distinct element method, is an analytical method which recognizes interactions between several types of solid particles; predicts the motion of enormous number of small particles or efficiency of the combination of more than two types of particles; or pre-measures the external forces on structures applied by particles. In other words, it is a Lagrangian method which decides the movement by using the six degrees of freedom motion equations and considers all forces of individual particles; and it uses to receive the time integration with explicit method.</span></p>
        <p style="margin-top: 0pt; margin-bottom: 0pt; text-indent: 36pt; line-height: 125%; font-size: 11pt;"><span style="font-family: Calibri; color: #3f3f3f; background-color: #ffffff;">This method analyses not only the mixing and movement of particles but also the effect and the influence over a toner that is affected by electromagnetic fields inside of a digital printer, also small particles like dust, and dust that moves on the space when there is airflow. Also, it includes particles like aerosol analysis that helps the interpretation of the behaviour in a small particle.</span></p>
        <p style="margin-top: 0pt; margin-bottom: 0pt; line-height: 125%; font-size: 11pt;"><span style="font-family: Calibri; color: #1d2129; background-color: #ffffff;"> Therefore, DEM requires a lot of memory and computing power for its very small-time step and enormous number of particles to be considered generally. </span><span style="font-family: Calibri; color: #1d2129; background-color: #ffffff;">S</span><span style="font-family: Calibri; color: #1d2129; background-color: #ffffff;">amadii/dem, which is designed to perform analysis based on the GPU and parallel processing techniques, supplies reliable analysis results by analysing a variety of large-scale grain boundary issues at a high speed.</span> <span style="font-family: Calibri; color: #3f3f3f; background-color: #ffffff;">To calculate the movement and force of particles requires much more calculations than we think.</span></p>
        <h2 style="margin-top: 2pt; margin-left: 43pt; margin-bottom: 0pt; text-indent: -28.8pt; page-break-inside: avoid; page-break-after: avoid; font-size: 13pt;"><span style="font-family: Calibri; font-weight: normal; color: #2f5496;">5.2</span> <a name="_Toc81323180"></a><a name="_Toc81330562"></a><a name="_Toc81390114"></a><a name="_Toc81411727"></a><a name="_Toc81413124"></a><a name="_Toc81420235"></a><a name="_Toc81498995"></a><a name="_Toc85794199"></a><a name="_Toc85794526"></a><a name="_Toc85798830"></a><a name="_Toc97543049"></a><a name="_Toc98784282"></a><span style="font-family: Calibri; font-weight: normal; color: #2f5496;">Samadii-</span><span style="font-family: Calibri; font-weight: normal; color: #2f5496;">Dem</span> <span style="font-family: Calibri; font-weight: normal; color: #2f5496;">on Azure Platform</span></h2>
