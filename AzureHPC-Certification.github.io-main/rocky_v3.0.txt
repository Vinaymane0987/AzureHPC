            <h3 style="margin-top: 0pt; margin-bottom: 14pt; background-color: #ffffff;"><a name="_Introduction"></a><span style="font-family: 'Segoe UI'; color: #171717;">Deploy Ansys Rocky for HPC on a virtual machine</span></h3>
            <p style="margin-top: 0pt; margin-bottom: 8pt; line-height: 150%; font-size: 11pt;"><span style="font-family: 'Segoe UI'; ">This article briefly describes the steps for running</span> <a style="text-decoration: none;" href="https://www.ansys.com/en-in/products/fluids/ansys-rocky"><span style="font-family: 'Segoe UI'; text-decoration: underline; color: #0000ff; background-color: #ffffff;">Ansys&rsquo;s</span></a><span style="font-family: 'Segoe UI';  "> Rocky application on a virtual machine (VM) that's deployed on Azure. It also presents the performance results of running </span><span style="font-family: 'Segoe UI';  ">Ansys Rocky</span><span style="font-family: 'Segoe UI';  "> on Azure.</span></p>
            <p style="margin-top: 0pt; margin-bottom: 8pt; line-height: 150%; font-size: 11pt;"><span style="font-family: 'Segoe UI';">Ansys Rocky</span><span style="font-family: 'Segoe UI';"> is a powerful 3D Discrete Element Modelling (DEM) Particle Simulation Software that quickly and accurately simulates the flow behaviour of bulk materials with complex particle shapes and size distributions, for typical applications such as conveyor chutes, mills, mixers, and other materials handling equipment.</span></p>
            <p style="margin-top: 0pt; margin-bottom: 8pt; line-height: 150%; background-color: #ffffff;"><span style="font-family: 'Segoe UI'; font-size: 11pt;">Ansys Rocky</span><span style="font-family: 'Segoe UI'; font-size: 11pt;"> is designed to s</span><span style="font-family: 'Segoe UI'; font-size: 11pt;">olve engineering problems. It</span><span style="font-family: 'Segoe UI'; font-size: 11pt;"> is uniquely capable to models real particle shapes including any solids, 2D shells, and rigid and flexible fibres. The simulations are fast and accurate. With multi-graphics processing unit (GPU) solver technology, you can simulate the behaviour of different shaped and sized particles in many industrial applications. Leading companies in the heavy equipment, off-road, mining, steelmaking, agriculture equipment manufacturing and food</span><span style="font-family: 'Segoe UI'; font-size: 11pt;"> processing industries use Ansys Rocky</span><span style="font-family: 'Segoe UI'; font-size: 11pt;"> to understand and predict granular material behaviours, evaluate equipment performance, and optimize processes</span><span style="font-family: 'Segoe UI'; font-size: 11pt;">.</span></p>
            <p style="margin-top: 0pt; margin-bottom: 6pt; line-height: 108%; font-size: 11pt;"><span style="font-family: 'Segoe UI'; font-weight: bold;">Simulate Real-Scale Experiments</span></p>
            <p style="margin-top: 0pt; margin-bottom: 0pt; line-height: 108%; font-size: 11pt;"><span style="font-family: 'Segoe UI';">Rocky DEM&rsquo;s high-performance computing (HPC) abilities can simulate a large number of real-sized and real shaped particles faster and with less memory usage.</span></p>
            <p style="margin-top: 6pt; margin-bottom: 6pt; line-height: 108%; font-size: 11pt;"><span style="font-family: 'Segoe UI'; font-weight: bold;">Model Realistic Materials</span></p>
            <p style="margin-top: 0pt; margin-bottom: 0pt; line-height: 108%; font-size: 11pt;"><span style="font-family: 'Segoe UI';">Modify a default shape, scan a real 3D object or custom design your own shape and import it. Then define the behaviour to be rigid or flexible, sticky or dry, breakable and more.</span></p>
            <p style="margin-top: 6pt; margin-bottom: 6pt; line-height: 108%; font-size: 11pt;"><span style="font-family: 'Segoe UI'; font-weight: bold;">Solve Unique Problems</span></p>
            <p style="margin-top: 0pt; margin-bottom: 0pt; line-height: 108%; font-size: 11pt;"><span style="font-family: 'Segoe UI';">Rocky DEM&rsquo;s representation of application-specific particle shapes is becoming essential to the successful engineering of bulk materials handling equipment.</span></p>
            <p style="margin-top: 0pt; margin-bottom: 0pt; font-size: 11pt; background-color: #ffffff;"><span style="font-family: 'Segoe UI';">To test </span><span style="font-family: 'Segoe UI';">the </span><span style="font-family: 'Segoe UI';">performance of</span><span style="font-family: 'Segoe UI';"> Ansys-Rocky</span><span style="font-family: 'Segoe UI';"> application</span><span style="font-family: 'Segoe UI';"> on Azure Platform,</span> <a style="text-decoration: none;" href="https://docs.microsoft.com/en-us/azure/virtual-machines/nda100-v4-series"><span style="font-family: 'Segoe UI'; text-decoration: underline; color: #0000ff;">ND A100 v4-series</span></a> <span style="font-family: 'Segoe UI';">and </span><a style="text-decoration: none;" href="https://docs.microsoft.com/en-us/azure/virtual-machines/nc-a100-v4-series"><span style="font-family: 'Segoe UI'; text-decoration: underline; color: #0000ff;">NC A100 v4-series</span></a> <span style="font-family: 'Segoe UI';">virtual </span><span style="font-family: 'Segoe UI';">machine is </span><span style="font-family: 'Segoe UI';">deployed.</span></p>

            <h3 style="margin-top: 14pt; margin-bottom: 14pt; font-size: 18pt;"><span style="font-family: 'Segoe UI';font-weight: bold;">Results and Recommendations</span></h3>
            <ol style="margin: 0pt; padding-left: 0pt;" type="1">
            <li style="margin-top: 14pt; margin-left: 31.16pt; text-align: justify; line-height: 150%; padding-left: 4.54pt; font-family: 'Segoe UI'; font-size: 12pt;">Ansys-Rocky Application is successfully deployed and tested on ND A100v4 and <span style="color: #171717; background-color: #ffffff;">NC A100</span><span style="color: #171717; background-color: #ffffff;">v4 </span>series Azure Virtual Machine</li>
            <li style="margin-left: 31.16pt; text-align: justify; line-height: 150%; padding-left: 4.54pt; font-family: 'Segoe UI'; font-size: 12pt;">The expected speed up is achieved when compared between CPU and one GPU.</li>
            <li style="margin-left: 31.31pt; margin-bottom: 14pt; text-align: justify; line-height: 150%; padding-left: 4.69pt; font-family: 'Segoe UI'; font-size: 12pt;">For all models, there is an optimal amount of computational hardware that achieves best price/performance, after which adding additional hardware does not scale the performance substantially. For this particular harvester model, we see the peak optimal performance achieved when we utilized 3 GPUs. For larger or more complex models, we expect this optimal GPU number to be higher, and for smaller ones we expect it to be lower.</li>
            </ol>
